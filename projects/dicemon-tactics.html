Dicemon Tactics — Nanuq Studio

Dicemon Tactics, ”Dicemon” gacha monster-collector for mobile — comes with beautiful gacha shop and summoning cutscenes, the progression loop (level-up, transmute, star promotion), and the live-service system (realtime chat, season pass, mail, and animated post-match chest rewards).

01

Gameplay & progression

  • Dicemon skills — Skills implemented on a node-graph sequence skill system, extending it with new graph nodes for new skill behaviours
  • Summoning cutscene — The gacha-opening sequence built as a state machine paired with Unity Timeline, with rarity-based VFX variants.
  • Gacha shop — Gacha shop system which handle addressable assets and cutscene router
  • Dicemon level-up — Dicemon progression system, by spending XP can result in stronger Dicemon's skill and stronger gameplay builds
  • Transmute — Converting system that transform Dicemon Units into upgrade materials that can be using in Dicemon progression system.
  • Star promotion — Sacrificing Dicemon to upgrade another Dicemon's star in Dicemon progression system.
02

Live-service & social

  • Chat — Multi-channel realtime chat (channel, direct, club, and announcements) over a WebSocket backend.
  • Post-match chest rewards — Animated chest-opening flow (state machine + Unity Timeline) with rarity upgrades.
  • Season pass — Free and premium tracks, XP-driven levels, reward claims, and paid level skips.
  • Mail — Server-driven mail box providing gifts and announcements.
